## sdt env merge

Merge multiple .env files (last value wins)

```
sdt env merge [flags]
```

### Options

```
      --files string    Comma-separated list of .env files to merge
  -h, --help            help for merge
      --output string   Write merged output to file (default: print JSON)
```
